# 📚 講座自動化システム - Web管理画面

## 🎨 概要

コマンドラインの代わりに、**ブラウザで使える美しい管理画面**です！
URLの入力から講座プロンプト生成まで、すべてGUIで簡単に操作できます。

## ✨ 特徴

- 🎨 **カラフルで見やすいUI**: グラデーションとアイコンで視覚的に分かりやすい
- 📊 **リアルタイム進行状況**: 処理の進み具合が一目で分かる
- 💾 **プロジェクト管理**: 複数の講座を管理・保存・再利用
- 🔄 **ワンクリック実行**: 複雑なコマンドを覚える必要なし
- 📈 **結果の可視化**: 品質評価や用語分析を見やすく表示
- 💻 **ブラウザで完結**: インストール不要、どこでもアクセス

## 🚀 起動方法

### 1. ターミナルを開く

### 2. 作業ディレクトリに移動

```bash
cd "/Users/apple/Desktop/クロスリンク様/ラーニング講座/講座自動化ver1"
```

### 3. Streamlitアプリを起動

```bash
streamlit run app.py
```

または

```bash
python3 -m streamlit run app.py
```

### 4. ブラウザが自動的に開きます

デフォルトURL: `http://localhost:8501`

## 📖 使い方（完全ガイド）

### ステップ0: プロジェクト作成

1. **左サイドバー**を開く
2. **「➕ 新規プロジェクト作成」**をクリック
3. プロジェクト名を入力（例: "ChatGPT講座_2024"）
4. **「作成」**ボタンをクリック

✅ プロジェクトが作成され、自動的に選択されます

---

### ステップ1: URL入力

**タブ: 🔗 ステップ1: URL入力**

#### Web記事のURLを追加

1. **「WebのURLを入力」**欄にURLを入力
   ```
   https://ja.wikipedia.org/wiki/ChatGPT
   ```
2. **「➕ 追加」**ボタンをクリック
3. 3〜5件のURLを追加（繰り返し）

#### YouTube動画のURLを追加

1. **「YouTubeのURLを入力」**欄にURLを入力
   ```
   https://www.youtube.com/watch?v=動画ID
   ```
2. **「➕ 追加」**ボタンをクリック
3. 1〜3件のURLを追加（繰り返し）

#### 削除したい場合

- 各URLの右側にある **🗑️ ボタン**をクリック

---

### ステップ2: リサーチ実行

**タブ: 🔍 ステップ2: リサーチ実行**

1. URL入力が完了したら、**「🚀 リサーチ開始」**ボタンをクリック
2. 進行状況バーが表示されます
3. 完了すると、以下が表示されます：
   - ✅ リサーチ完了メッセージ
   - 📊 統計情報（総情報源、総文字数、成功率）
   - 📄 実行ログ（展開可能）

💡 **完了後の確認**:
```
data/outputs/ ディレクトリに以下のファイルが生成されます：
- {プロジェクト名}_web.json
- {プロジェクト名}_youtube.json
- {プロジェクト名}_summary.json
```

---

### ステップ3: 品質・用語分析

**タブ: 📊 ステップ3: 品質・用語分析**

#### 品質検証

1. **「🔍 品質検証開始」**ボタンをクリック
2. 完了すると、以下が表示されます：
   - 🌟/✅/⚠️/❌ 総合品質評価
   - 📊 統計（総情報源、データポイント、信頼性の高い情報源）
   - 💡 推奨事項（展開可能）

#### 用語分析

1. （オプション）**講座テーマ**を入力
   ```
   例: ChatGPT業務活用
   ```
2. **「📖 用語分析開始」**ボタンをクリック
3. 完了すると、以下が表示されます：
   - 📊 統計（ユニーク用語数、トップ用語数）
   - 🔝 重要用語トップ10（表形式）
   - 💡 推奨事項（展開可能）

---

### ステップ4: 講座設定

**タブ: ⚙️ ステップ4: 講座設定**

フォームに以下を入力します：

| 項目 | 説明 | 例 |
|------|------|-----|
| **講座名** | ドロップダウンから選択 | ChatGPT業務活用の基本 |
| **ユニット番号** | （オプション）特定のユニットのみ生成 | 1 |
| **受講者像** | 誰が受講するか | ChatGPTを業務で使いたいビジネスパーソン |
| **到達目標** | 受講後に何ができるようになるか | ChatGPTを適切に活用して業務効率を向上できる |
| **想定時間** | ドロップダウンから選択 | 30分 |
| **トーン＆マナー** | ドロップダウンから選択 | 親しみやすく、実践的なトーン |

入力完了後、**「💾 設定を保存」**ボタンをクリック

---

### ステップ5: プロンプト生成

**タブ: 📝 ステップ5: プロンプト生成**

1. **「⚙️ 設定内容を確認」**で内容を確認（展開可能）
2. **「🚀 プロンプト生成」**ボタンをクリック
3. 生成が完了すると、以下が表示されます：
   - ✅ プロンプト生成完了メッセージ
   - 📄 生成されたプロンプト（スクロール可能）
   - 💾 ダウンロードボタン
   - 📋 コピーボタン
   - 🤖 Geminiで開くボタン

#### プロンプトの使用方法

**方法1: ダウンロード**
1. **「💾 ダウンロード」**ボタンをクリック
2. ファイルがダウンロードされます
3. Geminiにアップロードまたはコピペ

**方法2: 直接コピー**
1. プロンプト表示エリアをクリック
2. Ctrl+A (全選択) → Ctrl+C (コピー)
3. Gemini (https://gemini.google.com/) を開く
4. Ctrl+V (貼り付け)
5. 送信

**方法3: Geminiで開く**
1. **「🤖 Geminiで開く」**ボタンをクリック
2. 新しいタブでGeminiが開きます
3. プロンプトをコピーして貼り付け

---

## 💾 プロジェクト管理

### プロジェクトの保存

**左サイドバー**の **「💾 プロジェクト保存」**ボタンをクリック

保存内容:
- 入力したURL（Web + YouTube）
- 講座設定（受講者像、到達目標など）

### プロジェクトの読み込み

1. **左サイドバー**の **「既存プロジェクト」**セクション
2. ドロップダウンからプロジェクトを選択
3. 自動的に読み込まれます

### プロジェクトの場所

```
data/projects/{プロジェクト名}.json
```

## 🔧 トラブルシューティング

### アプリが起動しない

**エラー**: `streamlit: command not found`

**解決策**:
```bash
pip3 install streamlit
```

または

```bash
python3 -m pip install streamlit
```

### ポートが使用中

**エラー**: `Address already in use`

**解決策**: 別のポートで起動
```bash
streamlit run app.py --server.port 8502
```

### ブラウザが自動で開かない

**解決策**: 手動でブラウザを開く
```
http://localhost:8501
```

### データが消えた

**原因**: プロジェクトを保存していない

**解決策**:
- 左サイドバーの **「💾 プロジェクト保存」**を定期的にクリック
- `data/projects/` フォルダをバックアップ

## 💡 Tips & Tricks

### より良い結果を得るために

1. **Web記事**
   - 公式サイト、Wikipedia、大手メディアを選ぶ
   - 最新の情報（1年以内）を優先
   - 3〜5件が最適（多すぎても効果は薄い）

2. **YouTube動画**
   - 字幕付き動画を選ぶ
   - 10分以上の解説動画が理想
   - 1〜3件が最適

3. **講座設定**
   - 受講者像は具体的に（「ビジネスパーソン」より「ChatGPTを初めて使う営業担当者」）
   - 到達目標は行動ベースで（「理解する」より「実践できる」）

### キーボードショートカット

- `Ctrl + R`: ページを再読み込み
- `Ctrl + Shift + R`: キャッシュをクリアして再読み込み
- `Ctrl + K`: サイドバーの表示/非表示

## 📞 サポート

### よくある質問

**Q: プロジェクトはどこに保存される？**
A: `data/projects/` ディレクトリに保存されます。

**Q: 複数のプロジェクトを同時に編集できる？**
A: できません。1つずつ選択して作業してください。

**Q: プロンプトが長すぎてGeminiに貼り付けられない**
A: URL数を減らすか、ユニット番号を指定して部分的に生成してください。

**Q: アプリを終了するには？**
A: ターミナルで `Ctrl + C` を押します。
